# Product Charter

## Product Name

Work Portal for Senior Researchers

## Vision

Build a simple and readable office portal for senior researchers who already use BAND, cloud drive, and external email, without forcing them to change existing tools.

## Core Users

- Senior researchers and professors
- Small clinics/labs/research institutes with no internal groupware

## Core Problems

- Work data is fragmented across BAND, drive, and email.
- Existing tools are used, but there is no unified dashboard.
- Current workflows are hard for non-technical users to navigate quickly.

## Value Proposition

- Keep current tools, add one clear workspace.
- Prioritize accessibility: bigger text, fewer clicks, straightforward actions.
- Reduce context switching while preserving official source systems.

## Product Principles

1. Integration-first, not replacement-first.
2. Readability over feature density.
3. Security and policy compliance by default.
4. Tenant-safe architecture from day 1.
5. Minimal storage and transparent data handling.
6. Keep onboarding and service connection simple: from sign-up to external service connection in the fewest practical steps.

## Non-Goals (Initial Phase)

- Building a new email provider.
- Replacing BAND or Google Drive as the source of truth.
- Heavy analytics or AI features before core workflow reliability.
